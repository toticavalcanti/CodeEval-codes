REVERSE WORDS

Write a program which reverses the words in an input sentence.

INPUT SAMPLE:

The first argument is a file that contains multiple sentences, one per line. Empty lines are also possible.

For example:

Hello World
Hello CodeEval

OUTPUT SAMPLE:

Print to stdout each sentence with the reversed words in it, one per line. Empty lines in the input should be ignored. Ensure that there are no trailing empty spaces in each line you print.

For example:

World Hello
CodeEval Hello

REVERTE PALAVRAS

Escrever um programa que reverta as palavras de uma sentença de entrada

ENTRADA DA AMOSTRA:

O primeiro argumento é um arquivo que contém várias frases, uma por linha. Linhas vazias também são possíveis.

Por exemplo:

Hello World
Hello CodeEval

SAÍDA DA AMOSTRA::

Imprimir na stdout cada frase com as palavras invertidas, uma por linha. Linhas vazias na entrada devem ser ignoradas. Certifique-se de que não existem espaços vazios à direita em cada linha de impressão.

Por exemplo:

World Hello
CodeEval Hello
