STRING PERMUTATIONS

CHALLENGE DESCRIPTION:

Write a program which prints all the permutations of a string in alphabetical order. We consider that digits < upper case letters < lower case letters. The sorting should be performed in ascending order.

INPUT SAMPLE:

Your program should accept a file as its first argument. The file contains input strings, one per line.

For example:
hat
abc
Zu6

OUTPUT SAMPLE:

Print to stdout the permutations of the string separated by comma, in alphabetical order.

For example:
aht,ath,hat,hta,tah,tha
abc,acb,bac,bca,cab,cba
6Zu,6uZ,Z6u,Zu6,u6Z,uZ6

PERMUTAÇÃO DE STRING

DESCRIÇÃO DO DESAFIO :

Escreva um programa que imprima todas as permutações de uma seqüência de caracteres em ordem alfabética. Consideramos que os dígitos < letras maiúsculas < letras minúsculas. A classificação deve ser realizada em ordem crescente.

ENTRADA DA AMOSTRA:

Seu programa deve aceitar um arquivo como seu primeiro argumento. O arquivo contém strings de entrada, uma por linha.

Exemplo de entrada:
hat
abc
Zu6

SAÍDA DA AMOSTRA:

Imprimir na stdout as permutações da seqüência de caracteres separados por vírgula, em ordem alfabética.

Exemplo de saída:
aht,ath,hat,hta,tah,tha
abc,acb,bac,bca,cab,cba
6Zu,6uZ,Z6u,Zu6,u6Z,uZ6
