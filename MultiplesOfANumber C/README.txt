MULTIPLES OF A NUMBER

CHALLENGE DESCRIPTION:
Given numbers x and n, where n is a power of 2, print out the smallest multiple of n which is greater than or equal to x. Do not use division or modulo operator.

INPUT SAMPLE:

The first argument will be a path to a filename containing a comma separated list of two integers, one list per line. E.g.

13,8
17,16

OUTPUT SAMPLE:

Print to stdout, the smallest multiple of n which is greater than or equal to x, one per line. E.g.

16
32

---------------------------------------------------------------------------------------------------------------------------------------------------------------------

MÚLTIPLOS DE UM NÚMERO

DESCRIÇÃO DO DESAFIO:
Dado números x e n, em que n é uma potência de 2, imprimir o menor múltiplo de N que é maior ou igual a x. Não use divisão ou operador de módulo.

ENTRADA DA AMOSTRA:

O primeiro argumento será um caminho para um arquivo que contém uma lista separada por vírgula de dois inteiros, uma lista por linha. Por exemplo.

13,8
17,16

Exemplo de saída:

Impressão na saída padrão, o menor múltiplo de N que é maior ou igual a x, um múltiplo em cada linha. Por exemplo.

16
32
