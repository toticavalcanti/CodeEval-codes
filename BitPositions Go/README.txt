BIT POSITIONS
CHALLENGE DESCRIPTION:

Given a number n and two integers p1,p2 determine if the bits in position p1 and p2 are the same or not. Positions p1 and p2 are 1 based.

INPUT SAMPLE:

The first argument will be a path to a filename containing a comma separated list of 3 integers, one list per line. E.g.

86,2,3

125,1,2

OUTPUT SAMPLE:

Print to stdout, 'true'(lowercase) if the bits are the same, else 'false'(lowercase). E.g.

true
false

-------------------------------------------------------------------------------------------------
POSIÇÕES DOS BITS
DESCRIÇÃO DO DESAFIO:

Dado um número n e dois inteiros P1, P2 determinar se os bits na posição P1 e P2 são iguais ou não. Posições P1 e P2 são baseadas em 1.

ENTRADA DA AMOSTRA:

O primeiro argumento será um caminho para um arquivo que contém uma lista separada por vírgula de 3 números inteiros, uma lista por linha. Por exemplo.

86,2,3

125,1,2

Exemplo de saída:

Impressão na saída padrão, 'true' (minúsculas) se os bits são iguais, ou 'false' (minúsculas) se são diferentes. Por exemplo.

true
false
